HTML and CSS Cheat Sheet

HTML (HyperText Markup Language)

1. Basics

HTML is the standard language for creating web pages.

Begin every HTML document with <!DOCTYPE html> to define the document type.

Wrap content inside <html> tags, with <head> for metadata and <body> for content.

2. HTML Elements

Elements are defined by tags.

To create a section, use a block-level element like <div>.

Use inline elements like <span> for smaller parts of text within a block.

3. Attributes

Attributes provide additional information about elements.

To add a unique identifier, use the id attribute (e.g., <div id="unique">).

For styling, use the class attribute to group elements (e.g., <div class="container">).

4. Headings

Headings define the structure of the document.

Use <h1> for the main title, and <h2> to <h6> for subheadings.

Maintain a logical order for better SEO and accessibility.

5. Paragraphs

Use the <p> tag to create paragraphs of text.

Keep paragraphs short for better readability.

Avoid using <br> for line breaks; use new paragraphs instead.

6. Comments

Comments help you annotate your code.

Write comments between <!-- and --> to make notes without affecting display.

7. Links

Create hyperlinks with the <a> tag.

Set the href attribute to the destination URL (e.g., <a href="https://example.com">).

Use descriptive text between the opening and closing <a> tags for better accessibility.

8. Images

Insert images using the <img> tag.

Specify the src attribute for the image source and alt for an alternative text description.

Ensure images are relevant to the content for a better user experience.

9. Head Section

Include metadata and links to stylesheets within the <head> section.

Use the <title> tag for the webpage title that appears in the browser tab.

Add <meta> tags for description and keywords for SEO purposes.

CSS (Cascading Style Sheets)

1. Syntax

CSS is used to style HTML elements.

Write CSS rules with a selector followed by curly braces.

Inside the braces, use properties and values (e.g., color: blue;).

2. Colors

Define colors using color names, hex codes, or RGB values.

Apply color styles by targeting specific elements in your CSS.

Use background color properties to enhance element visibility.

3. Backgrounds

Set background properties for elements, including color and images.

Use background-color to change the background and background-image to add an image.

Combine background properties for complex styling.

4. Text

Style text with properties like font-size, color, and text-align.

Use font-family to choose between serif and sans-serif fonts.

5. Fonts

Specify font styles with properties such as font-weight and font-style.

Utilize web fonts by linking to external font services in your <head>.

6. Links

Style links with pseudo-classes for different states (e.g., :hover, :active).

Define styles for links to improve user experience and accessibility.
